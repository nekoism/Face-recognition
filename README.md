# Face-recognition
Python day 6 : Face recognition

sesi pembelajaran membuat face recognition 

cara menjalankan.
saya menggunakan terminal alacritty 

-buat file untuk menyimpan face venv

-selanjutnya masuk kedalam file melalui terminal dan masukan : py -m venv namafile #untuk membuat virtual environment

-lalu masukkan : source namafile/Scripts/activate 

-lalu masukkan lagi diterminal py -m pip install opencv-python #untuk menginstall opencv (cv2) yang bertujuan untuk olah gambar dan vidio

-lalu jalankan face.py d terminal
